Get Started with HTML and CSS





HTML and CSS Structure



HyperText Markup Language (HTML) is a standard markup coding language used to design web documents. Cascading Style Sheets (CSS) may personalize the design and automate specific actions. Most web pages you access, whether they’re articles, social media platforms, or even online shopping sites, most web pages you access are usually coded in HTML.



HTML


HTML is formed using a mixture of tags, elements, attributes, and elements.


HTML tags are keywords that search the language database to find a pre-defined styling associated with the element. This means certain tags, such as the <h1> tag, used for the most crucial heading for your website, have pre-defined stylings to display the item. Although, later, you may also use CSS to restyle the object by changing factors such as font size, font type, font colour, and so much more.


There are hundreds of different HTML tags. Even very experienced web developers cannot memorize every single tag, so don’t worry if you don’t get the hang of it right away. This website provides a detailed list of each HTML tag and its purpose.


An HTML element is anything that is written inside of an HTML tag. An example would be the following:


<p> </p> would be the tag, and however, if you were to write a sentence inside the tag brackets, it would be classified as HTML elements.


There are two main types of HTML elements: block and inline. Block elements stretch as far as they can across the width of a web page and will begin on a new line for every new element. In comparison, inline elements are more generous and only take up minimal space on a web page to fit multiple elements.


Different HTML elements have default assigned values as either block or inline. However, with CSS, you may code elements to go from inline to block and vice versa.


Now that you know the basic setup of HTML let’s move on to its best friend, CSS.



CSS


When browsing the internet, you may have noticed that a majority of websites have the HTTPS prefix in the URL, while some may begin with HTTP. What does the extra “s” mean? Well, simply the “s” stands for SSL, which indicates that the website you are accessing is secure, which is much more preferred.





Diagram of a CSS Rule - https://www.digitalocean.com/community/tutorials/how-to-understand-and-create-css-rules



The selector defines which HTML element is to be styled, and curly braces encapsulate the property and value within. A selector can be an element, class name, or an id for different HTML content, such as h1, div, or #nav.


The property is the aspect of the HTML element that you would like to change with CSS. Some examples could be colour or background-color. There is an extensive catalogue of CSS properties. This website provides a complete list of these properties.


The value is the specific change that is being assigned to the property. Some examples could be 16px for font-size or #000 for color.
